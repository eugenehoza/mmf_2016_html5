Необходимо доработать игру Packman.
Исходный код - 06 js canvas\game.html

Свои изменения коммитить только в этот файл. Не создавать новые файлы!

Антонович - сделать 10 уровней для игры. Для каждого уровня зафиксировать сгенерированные массивы препятствий и еды. Подобрать оптимальные варианты данных для уровней. После съедания всей еды автоматичемки переходить на новый уровень. Выводить интегральной вермя, затраченное на прохождение всех уровней на экран.
Для Каждый следующий должен быть сложнее предыдущего
Бабицкая - поддержать поворот рта пакмана в сторону движения. Добавить вывод таймера минута-секунда-милисекунда на экран возле канваса. Добавить кнопки начать и завершить, которые будут перезапускать и останавливать таймер.
Гришко - добавить генератор злодеев. Добавить кнопку "создать злодеев" и 1 поле с настройкой - кол-во. После нажатия кнопки создавать массив объектов вида { x: 123, y: 124}. По созданным обхектам рисовать злодеев в виде кружков с красной заливкой фиксированного размера. Поддержать перемещение злодеев в произвольные стороны с интервалом 1 раз в секунду. Массив с злодеями не должен пересекаться с массивами еды и препядствий.
Делендик - добавить генератор еды. Добавить кнопку "создать еду" и 1 поле с настройкой - кол-во. После нажатия кнопки создавать массив объектов вида { x: 123, y: 124}. По созданным обхектам рисовать еду в виде кружков с зеленой заливкой фиксированного размера. Поддержать удаление элемента из массива и в случае прохождения пакмана через еду. Массив с едой не должен пересекаться с массивом с препядствиями
Дробуш - добавить генератор препятствий. Добавить кнопку создать препятствия и 3 поля с настройками - кол-во, минимальный размер, максимальный размер. После нажатия кнопки создавать массив объектов вида { x: 123, y: 124, size: 20}. По созданным обхектам рисовать препятствия в виде квадратов с серой заливкой и размером size. Поддержать невозможность прохождения пакмана через препядствия. Массив с препядствиями не должен пересекаться с массивом с едой
